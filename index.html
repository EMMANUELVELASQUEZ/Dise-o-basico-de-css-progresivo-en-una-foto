Diseño CSS Progresivo

Emmanuel Velasquez Geron

01

Imagen base

Se coloca la imagen con fondo transparente sobre un contenedor sin estilos.

styles.css — paso 1 
.demo {
  width: 260px;
  height: 300px;
  position: relative;
  overflow: hidden;
}

.foto {
  width: 100%;
  height: 100%;
  object-fit: cover;
  object-position: center top;
}
02

Fondo oscuro — background-color

Se aplica un color de fondo oscuro al contenedor. La imagen con fondo transparente se integra naturalmente.

styles.css — paso 2 
.demo {
  width: 260px;
  height: 300px;
  position: relative;
  overflow: hidden;

  /* Nuevo en este paso */
  background-color: #1a1a1a;
}
03

Textura de puntos — background-image

Se usa radial-gradient con background-size para crear el patron de cuadricula de puntos del fondo.

styles.css — paso 3 
.demo {
  width: 260px;
  height: 300px;
  position: relative;
  overflow: hidden;
  background-color: #1a1a1a;

  /* Nuevo en este paso */
  background-image: radial-gradient(
    1px 1px at center,
    #2d2d2d 0%,
    #2d2d2d 50%,
    transparent 50%
  );
  background-size: 18px 18px;
}
04

Llaves { } — ::before y ::after

Se usan los pseudo-elementos ::before y ::after del contenedor para dibujar las llaves sin agregar HTML extra.

styles.css — paso 4 
/* Nuevo en este paso */
.demo::before,
.demo::after {
  content: "{";
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  font-family: monospace;
  font-size: 80px;
  font-weight: 900;
  color: #39ff14;
  z-index: 1;
  line-height: 1;
}

.demo::before {
  content: "{";
  left: 4px;
}

.demo::after {
  content: "}";
  right: 4px;
}
05

Efecto glow — text-shadow

Se aplica text-shadow a los pseudo-elementos para dar el efecto de brillo neon verde caracteristico de la imagen.

styles.css — paso 5 
.demo::before,
.demo::after {
  content: "{";
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  font-family: monospace;
  font-size: 80px;
  font-weight: 900;
  color: #39ff14;
  z-index: 1;
  line-height: 1;

  /* Nuevo en este paso */
  text-shadow:
    0 0 10px #39ff14,
    0 0 25px #39ff14,
    0 0 50px #39ff1466;
}
06

Resultado final — border-radius

Se redondean las esquinas del contenedor con border-radius para terminar el diseno.

styles.css — resultado final 
.demo {
  width: 260px;
  height: 300px;
  position: relative;
  overflow: hidden;
  background-color: #1a1a1a;
  background-image: radial-gradient(
    1px 1px at center,
    #2d2d2d 0%, #2d2d2d 50%,
    transparent 50%);
  background-size: 18px 18px;

  /* Nuevo en este paso */
  border-radius: 16px;
}

.demo::before,
.demo::after {
  content: "{";
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  font-family: monospace;
  font-size: 80px;
  font-weight: 900;
  color: #39ff14;
  z-index: 1;
  line-height: 1;
  text-shadow:
    0 0 10px #39ff14,
    0 0 25px #39ff14,
    0 0 50px #39ff1466;
}

.demo::before { left: 4px; }
.demo::after  {
  content: "}";
  right: 4px;
}

.foto {
  position: relative;
  z-index: 2;
  width: 100%;
  height: 100%;
  object-fit: cover;
  object-position: center top;
}
